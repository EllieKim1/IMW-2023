Inspiration

Recently, I've been interested in 3D modeling, so I decided to create my portfolio website inspired by 3D fruits.

Initial sketches and concepts

I started sketching and designing my website layout and drafts through the website called Spline, recommended by prof.HA.

Refined concepts and in-process work

Although creating the plate was easier, as I continued arranging the fruits, it looked odd. So, I ended up making each fruit in Blender in 3D. Initially, I experimented with arranging the fruits in a floating configuration, but eventually reverted to a composition with fruits in a basket and a few floating in the air. It took quite some time...
